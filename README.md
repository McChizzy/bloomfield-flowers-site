# Bloomfield Flowers Site

Preview-ready Vite site for Bloomfield Flowers.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Vercel deployment

This project is ready for Vercel as a static Vite app.

### Recommended settings
- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

### Notes
- `vercel.json` is included for SPA route rewrites.
- Main preview image currently uses `/images/hero1.jpeg`.
- Catalog data is still placeholder until the final bouquet list is added.
